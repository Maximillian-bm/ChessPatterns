# Chess Pattern Generator

A simple Java project for generating visual patterns from chess-like piece movement rules.

You specify one or more pieces by giving the moves they can make and an optional color. The program then places those pieces step by step on a square board and renders the resulting pattern to a PNG file.

## How to use

Compile the project:

```bash
javac *.java
```

Run the program:

```bash
java main
```

Then use commands such as:

- `help` — show available commands
- `add 1,2 2,1 -1,-2 -2,-1 -1,2 -2,1 1,-2 2,-1 #000000`
- `list`
- `clear`
- `render 100 output.png`
- `exit`

If you omit the `add` command, you can also add a piece by supplying move items directly on a single line.

## Input Format

The input describes chess moves as a series of coordinate pairs, where each pair represents the relative change in position (delta row, delta column) that a piece can make from its current square. Each move is specified as `dx,dy`, where:

- `dx` is the change in row (positive for down, negative for up)
- `dy` is the change in column (positive for right, negative for left)

Moves are separated by spaces. Optionally, you can append a hex color code (e.g., `#000000` for black) at the end to specify the color of the pattern generated by this piece.

### Example: Knight Moves

A knight in chess can move in an L-shape: 2 squares in one direction and 1 square perpendicular. The 8 possible moves are:

- `1,2` (1 down, 2 right)
- `2,1` (2 down, 1 right)
- `-1,-2` (1 up, 2 left)
- `-2,-1` (2 up, 1 left)
- `-1,2` (1 up, 2 right)
- `-2,1` (2 up, 1 right)
- `1,-2` (1 down, 2 left)
- `2,-1` (2 down, 1 left)

This is represented in the input as: `add 1,2 2,1 -1,-2 -2,-1 -1,2 -2,1 1,-2 2,-1 #000000`

## Example piece definition

A knight-like piece:

```text
add 1,2 2,1 -1,-2 -2,-1 -1,2 -2,1 1,-2 2,-1 #000000
```

## Output

The `render` command produces a PNG image using the current piece definitions.
